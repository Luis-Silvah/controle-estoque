## :ledger: Projeto Integrador

## Configuração docker

Criação do container
````js
    docker-compose up --build --no-recreate -d
````

Atualizar o docker-compose
````js
    docker-compose up -d
````

Acessar o terminal para iniciar projeto
````js
    docker exec -it projeto_integrador sh
````
